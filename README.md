# AI Customer Support Assistant using RAG

## Overview

An AI-powered customer support automation workflow built using n8n, OpenRouter LLMs, and Pinecone.

The system automatically classifies incoming customer emails, retrieves relevant information from a knowledge base using Retrieval-Augmented Generation (RAG), and generates context-aware responses with minimal manual intervention.

## Features

* Automated customer email processing
* AI-powered email classification
* Retrieval-Augmented Generation (RAG)
* Pinecone vector database integration
* Context-aware response generation
* Gmail automation workflow
* API-based workflow orchestration
* Reduced manual support effort

## Tech Stack

* n8n
* OpenRouter / OpenAI Compatible Models
* Pinecone
* Gmail API
* REST APIs
* RAG

## Workflow

1. Customer email received
2. Gmail Trigger activates workflow
3. AI classifier identifies support requests
4. Relevant knowledge retrieved using RAG
5. AI Agent generates response
6. Workflow updates email status and labels
